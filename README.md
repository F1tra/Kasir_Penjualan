## fitur aplikasi kasir

1. Mencatat transaksi pembelian barang
2. Menampilkan daftar transaksi pembelian barang
3. Authentikasi login pengguna
4. Pengaturan data pengguna
5. Pengaturan data master produk

## cara instal ke lokal
-   siapkan folder kosong
-   git bash di folder kosong tadi
-   git clone https://github.com/F1tra/Kasir_Penjualan.git 
-   buka ke vscode atau sublem 
-   langung klick ctrl+' 
-   ketika sudah di terminal
-   cd nama folder KASIR_PENJUALAN
-   composer update/install
-   cv .env.example.env
-   php artisan key generate
-   php artisan db:seed
